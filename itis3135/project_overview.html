Project Overview — 49ers Car Culture Club (C3)

Client Category (Step 01): Student group at UNCC
Project Category (Step 02): Group site that encourages participation & communication

1) Application Purpose

The 49ers Car Culture Club (C3) site organizes events (meets, cruises, tech nights), recruits members, and shares media. It improves visibility, centralizes announcements, and offers light interactivity (gallery, FAQs, event highlights, read-only event list).

2) Intended Users

  • Prospective members: basics, how to join, upcoming events
  • Active members: dates, locations, quick RSVP links
  • Officers: publish schedules & rules (content provided)
  • Partners/Sponsors: view past events & contact

3) Content Overview

  • About/mission, officer bios, safety guidelines
  • Events list and highlights
  • Gallery with accessible lightbox
  • Join/Contact interest form + campus map
  • Resources (optional): policies & useful links

8) Page Design Details

Home (index.html)
  • Purpose: Welcome/value prop; highlight next event
  • Audience: Everyone
  • Content: Hero, 1–2 sentence mission, “Next Event” card, 3-image preview, Join CTA
  • Inputs: None
  • Controls: Buttons/links to Events & Join
  • Actions: “Details” opens modal or Events page
  • Notes: jQuery-UI Accordion for FAQs
About (about.html)
  • Purpose: Mission, expectations, officer bios
  • Audience: Prospects & partners
  • Inputs: None
  • Actions: Expandable safety checklist (Accordion)
Events (events.html)
  • Purpose: Central events list
  • Audience: Members & prospects
  • Content: Event cards (title, date, time, location, brief)
  • Inputs: None (read-only)
  • Controls: “Details” modal; “RSVP” → Google Form
  • Actions: AJAX (read-only) to load a static JSON of events
Gallery (gallery.html)
  • Purpose: Showcase photos
  • Inputs: None (no uploads per course rules)
  • Controls: Lightbox with keyboard navigation
Join & Contact (join.html)
  • Purpose: Collect interest & questions
  • Inputs: Name, UNCC email, grad year, optional car info, interests, message
  • Validations: Required fields, email pattern, length limits
  • Actions: Prefilled mailto: or Google Form
  • Notes: Embedded Google Map to meet spot
Resources (resources.html) — optional
  • Purpose: Campus policies, track days, safety guides
  • Actions: External links open in new tabs

9) Planned Interactivity (JavaScript/jQuery/AJAX)

  • jQuery-UI Accordion for FAQs/Safety
  • Gallery Lightbox (jQuery plugin)
  • Events Loader via fetch of static JSON (read-only)
  • Form Validations (HTML5 + JS)
  • Map Embed (Google Maps iframe)

10) Interaction References

  • University club pages with clear CTAs & event lists
  • Accessible lightbox galleries
  • Read-only event feeds generated from static JSON
